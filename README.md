# ASK Modulation & Demodulation System

This project demonstrates Amplitude Shift Keying (ASK), a digital modulation technique where binary data is transmitted by varying signal amplitude. It also includes a simple demodulation approach using envelope detection.

## Features
- ASK-based digital modulation  
- Binary data transmission using amplitude variation  
- Basic signal demodulation concept  
- Simulation of wireless communication principles  

## Hardware Required
- Arduino board  
- LED or transmitter circuit  
- Analog input receiver setup  

## Applications
- Wireless communication systems  
- RF transmission concepts  
- Embedded communication learning
